Previous versions of the site included always-on social media handles across the top of each page, just below the header.

Semantic HTML & friends

Articles are sections, and sections are articles! You won't notice a visual change, but if you inspect the site you'll see an improvement to the semantics of the HTML structure overall.

It feels good putting some of the new information in my brain to use in a practical setting.

When you scroll through the site using buttons, it's smoother now! I also added more buttons so you can zoom around more playfully.

Now that the design is (mostly) figured out, I can start adding fun features and make the site more dynamic. Look forward to more graphics and interactivity as I continue to design each page
